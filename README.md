
## Installation
Assuming python and `pip` are installed.
1. Install `pipenv`
```bash
pip install pipenv
```
2. Start a `pipenv` envrionment. Ensure you are in the folder with `Pipfile`.
```bash
pipenv shell
pipenv install
```

## Usage
1. Start up the Flask app
```bash
flask run
```
2. Navigate to http://127.0.0.1:5000/
